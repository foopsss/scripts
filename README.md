# Scripts
Here I store some scripts I made thanks to the Internet community that I'd like to share.

* In the **ADB** folder, there's a script I made both for Batch and Bash that provides a simple TUI to use ADB and Fastboot. Both are completely functional, although the Bash version is definitely the one I have maintained more in recent times.
* In the **Automatización** folder, there are some Bash scripts I use to automate tasks on my Gentoo (and previously on Fedora Silverblue) system.
* In the **Backups** folder there are some PowerShell and Bash scripts I use to make file backups on my devices.
* In the **Calibri** folder, there's a Bash script to install the Calibri font family for the local user. However, the font files **are not included**. It can easily be tweaked for other font families.

Credits for all the websites and people I learnt something from while making these scripts are included in the CREDITS.md file.
